# AppLock Privacy Policy

This repository hosts the privacy policy for the AppLock Android application.

## Repository Description

**AppLock Privacy Policy - Comprehensive privacy documentation for Android app locking application. Includes detailed information about permissions, data collection, storage, and user rights. Hosted via GitHub Pages for easy access and transparency.**

## Setup Instructions

1. **Create a GitHub repository** named `applock-privacy-policy` (or your preferred name)

2. **Upload the privacy policy file:**
   - Upload `privacy-policy.html` to the repository
   - You can rename it to `index.html` if you want it to be the default page

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click on **Settings** → **Pages**
   - Under **Source**, select:
     - **Deploy from branch**
     - Branch: `main` (or `master`)
     - Folder: `/ (root)`
   - Click **Save**

4. **Access your privacy policy:**
   - Your privacy policy will be available at: `https://[your-username].github.io/applock-privacy-policy/`
   - Or if you used `index.html`: `https://[your-username].github.io/applock-privacy-policy/`

## Repository Structure

You can customize the privacy policy by:
